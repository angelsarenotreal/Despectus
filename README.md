# Despectus

A dark themed League of Legends dashboard.

It automatically connects to the League client and displays:
- Your profile
- Current Solo/Duo rank, LP, winrate, and progression
- Last 10 ranked games with KDA, CS, and match duration
- Most-played champions over the last 10 games

---

## Download
Go to the **Releases** page and download `Despectus-Setup.exe`.

## Install
Run the installer and follow the steps.

## First Launch
On first run, the app will ask for a Riot API Key.
